## Overview

The Postgresql connector allows you to sync data from Postgresql into Airweave, making it available for search and retrieval by your agents.

## Configuration


### PostgreSQLSource

PostgreSQL source implementation.

This source connects to a PostgreSQL database and generates entities for each table
in the specified schemas. It uses database introspection to:
1. Discover tables and their structures
2. Create appropriate entity classes dynamically
3. Generate entities for each table's data

#### Authentication

This connector uses **Configuration-based authentication**.

Authentication configuration class: `PostgreSQLAuthConfig`

PostgreSQL authentication configuration.

The following configuration fields are required:

| Field | Type | Description | Required |
|-------|------|-------------|----------|
| host | str | The host of the PostgreSQL database | Yes |
| port | int | The port of the PostgreSQL database | Yes |
| database | str | The name of the PostgreSQL database | Yes |
| user | str | The username for the PostgreSQL database | Yes |
| password | str | The password for the PostgreSQL database | Yes |
| schema | str | The schema of the PostgreSQL database | No |
| tables | str | Comma separated list of tables to sync. For example, 'users,orders'. For all tables, use '*' | No |
