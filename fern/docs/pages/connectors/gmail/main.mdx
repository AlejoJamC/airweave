## Overview

The Gmail connector allows you to sync data from Gmail into Airweave, making it available for search and retrieval by your agents.

## Configuration


### GmailSource

Gmail source implementation (read-only).

Retrieves and yields Gmail objects (labels, threads, messages, drafts)
as entity schemas defined in entities/gmail.py.

#### Authentication

This connector uses **OAuth 2.0 with refresh token**.

This connector uses OAuth authentication. You can connect through the Airweave UI, which will guide you through the OAuth flow.

| Field | Type | Description |
|-------|------|-------------|
