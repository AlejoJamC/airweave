## Overview

The Oracle connector allows you to sync data from Oracle into Airweave, making it available for search and retrieval by your agents.

## Configuration


### OracleSource

Oracle source implementation.

This source connects to an Oracle database and generates entities for each table
in the specified schemas. It uses database introspection to:
1. Discover tables and their structures
2. Create appropriate entity classes dynamically
3. Generate entities for each table's data

#### Authentication

This connector uses **Configuration-based authentication**.

Authentication configuration class: `OracleAuthConfig`

Oracle authentication configuration.

The following configuration fields are required:

| Field | Type | Description | Required |
|-------|------|-------------|----------|
| host | str | The host of the Oracle database | Yes |
| port | int | The port of the Oracle database | Yes |
| database | str | The name of the Oracle database | Yes |
| user | str | The username for the Oracle database | Yes |
| password | str | The password for the Oracle database | Yes |
| schema | str | The schema of the Oracle database | No |
| tables | str | Comma separated list of tables to sync. For example, 'users,orders'. For all tables, use '*' | No |
