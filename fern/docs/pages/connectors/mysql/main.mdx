## Overview

The Mysql connector allows you to sync data from Mysql into Airweave, making it available for search and retrieval by your agents.

## Configuration


### MySQLSource

MySQL source implementation.

This source connects to a MySQL database and generates entities for each table
in the specified schemas. It uses database introspection to:
1. Discover tables and their structures
2. Create appropriate entity classes dynamically
3. Generate entities for each table's data

#### Authentication

This connector uses **Configuration-based authentication**.

Authentication configuration class: `MySQLAuthConfig`

MySQL authentication configuration.

The following configuration fields are required:

| Field | Type | Description | Required |
|-------|------|-------------|----------|
| host | str | The host of the MySQL database | Yes |
| port | int | The port of the MySQL database | Yes |
| database | str | The name of the MySQL database | Yes |
| user | str | The username for the MySQL database | Yes |
| password | str | The password for the MySQL database | Yes |
| schema | str | The schema of the MySQL database | No |
| tables | str | Comma separated list of tables to sync. For example, 'users,orders'. For all tables, use '*' | No |
